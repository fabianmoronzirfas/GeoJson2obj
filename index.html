Geojson2obj

Create Wavefront OBJ from GeoJson

Download .zip Download .tar.gz View on GitHub

THIS IS A GEOJSON TO WAVEFONT OBJ PARSER WRITTEN IN PROCESSING USING THIS JSON LIBRARY FROM AGORANSSON AND THIS OBJ LIBRARY BY N-E-R-V-O-U-S

it is used to transform GEOJson files to .obj files for usage in Adobe After Effects using the AEMap.jsx and The-Geo-Beast.
